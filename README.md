# 다솜 주점 주문 페이지
주점에서 사용할 수 있는 모바일 최적화 주문 페이지를 개발해보았습니다. Trello Board랑 연동했고, 로그인 안하고 카드 추가 가능한지는 테스트 전입니다.

## 주문 시스템
페이지에서 주문 정보를 입력하고 전송 버튼을 클릭하면 Trello의 지정된 Board에 카드가 추가됩니다. 주점 운영자들은 Trello Board를 이용해서 주문 내역을 간편하게 관리할 수 있습니다.
